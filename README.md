Многопоточная CLI-утилита для проверки доступности веб-сайтов

Особенности
- Конкурентность: Каждый сайт проверяется в отдельной горутине.
- Каналы: Используются для безопасной передачи данных между горутинами и синхронизации результатов.
- Скорость: Время выполнения равно времени ответа самого медленного сайта, а не сумме всех ответов.

Технический стек
- Language: Go (Golang)
- Concurrency: Goroutines, Channels (unbuffered)
- Network: `net/http` client

// Основной паттерн конкурентности в проекте:
